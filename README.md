## 🌐 About Me

I’m a Software Engineer with a degree in Software Technologies Engineering from FIME (Facultad de Ingeniería Mecánica y Eléctrica). I’m 25 years old and currently focused on building robust, scalable, and maintainable systems across both backend and frontend environments.

My strongest expertise lies in backend development using Node.js, JavaScript, and Progress 4GL, as well as frontend development with WebSpeed, SAP technologies, and modern UI frameworks. I also work extensively with SQL and NoSQL databases, API design, systems integration, and process automation within manufacturing and enterprise environments.

I enjoy solving structural, real-world problems—whether that means designing a clean data flow, optimizing a production pipeline, or building internal applications that empower entire teams.

I currently work at Viakable in the IT department as a Senior Developer. &nbsp; &nbsp;
